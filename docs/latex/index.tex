Q\+S\+ER is a application and hardware shield interface to manage and fully control a engine motor (Quanser Robots) on Galileo v2 shield. On this repository you will find the shield design under {\ttfamily shield} and the resources under {\ttfamily src}.



\doxysection*{Build}

You can build using the root makefile or you can use the cmake under cmake-\/build-\/release.


\begin{DoxyCode}{0}
\DoxyCodeLine{-\/ Compile the full project with `make`.}
\DoxyCodeLine{-\/ Clean built files with `make clean`.}
\end{DoxyCode}
