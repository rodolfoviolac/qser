QSER

QSER is a application and hardware shield interface to manage and fully control a engine motor (Quanser Robots) on Galileo v2 shield. On this repository you will find the shield design under shield and the resources under src.

Build

You can build using the root makefile or you can use the cmake under cmake-build-release.

- Compile the full project with `make`.
- Clean built files with `make clean`.
